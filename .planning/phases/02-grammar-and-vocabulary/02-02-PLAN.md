FIRST: Read data/vocabulary/all.json (first 5 entries) to confirm exact field names: latin, english, part_of_speech, declension, conjugation, gender, topics, frequency_rank.

SECOND: Create js/vocabulary.js as a self-contained vanilla JS file. Do NOT modify js/app.js.

Required structure:

```javascript
'use strict';

document.addEventListener('DOMContentLoaded', () => {
  const app = document.getElementById('app');
  const filters = getFilters();

  fetch('data/vocabulary/all.json')
    .then(r => { if (!r.ok) throw new Error('vocab'); return r.json(); })
    .then(words => {
      const filtered = applyFilters(words, filters);
      renderVocabulary(app, filtered, filters, words.length);
    })
    .catch(err => {
      app.innerHTML = '<h1>Vocabulary</h1><p class="error">Could not load vocabulary list. Please reload.</p>';
      console.error('[CE Latin] Vocabulary load error:', err);
    });
});
```

Implement these functions:

**getFilters():**
```javascript
function getFilters() {
  const params = new URLSearchParams(window.location.search);
  return {
    topic: params.get('topic'),           // decoded automatically — "war/army" not "war%2Farmy"
    freq: params.get('freq') ? parseInt(params.get('freq'), 10) : null
  };
}
```

**applyFilters(words, filters):**
- Start with `let result = [...words]`
- If filters.topic is set: `result = result.filter(w => w.topics.includes(filters.topic))`
- If filters.freq is set: sort result by frequency_rank ascending, then `result = result.slice(0, filters.freq)`
- Return result

**renderVocabulary(app, filtered, filters, total):**
- Clear app: `app.innerHTML = ''`
- Append h1: "Vocabulary" (or "Vocabulary — [Topic]" if topic filter active — capitalise first letter)
- Append filter nav (call renderFilterNav(filters))
- Append p.vocab-count: e.g. "211 words" or "12 words (family)" or "Top 50 by frequency"
- If filtered.length === 0: append p.vocab-empty with text "No words found for this filter."
- Otherwise: create div.vocab-list, iterate filtered calling renderWordItem(word) and appending each, then append vocab-list to app

**renderFilterNav(filters):**
- Create nav.vocab-filters
- Create "All words" link: `<a href="vocabulary.html" class="vocab-filter-link">All words</a>` — add class "active" if no topic and no freq filter
- Create topic links (one per topic — use this exact list to match the JSON topic values):
  - "family" → label "Family"
  - "war/army" → label "War &amp; Army" — href must be `vocabulary.html?topic=war%2Farmy` (encode the slash)
  - "gods" → label "Gods"
  - "travel" → label "Travel"
  - "daily-life" → label "Daily Life" — href: `vocabulary.html?topic=daily-life`
  - "nature" → label "Nature"
- Add class "active" to the link matching the current filters.topic
- Create frequency links:
  - "Top 50" → href `vocabulary.html?freq=50`
  - "Top 100" → href `vocabulary.html?freq=100`
- Add class "active" to the matching freq link if filters.freq is 50 or 100
- Return the nav element

**renderWordItem(word):**
- Create div.vocab-item
- Create span.vocab-latin with word.latin (use textContent — macrons must not be entity-encoded)
- Create span.vocab-english with word.english
- Create span.vocab-meta with buildMeta(word)
- Append all three spans to the div
- Return the div

**buildMeta(word):**
- Returns a string like "noun, f., 1st decl." or "verb, 1st conj." or "adj." depending on part_of_speech
- For nouns: `${word.part_of_speech}, ${word.gender}., ${word.declension} decl.`
- For verbs: `${word.part_of_speech}, ${word.conjugation ? word.conjugation + ' conj.' : 'irreg.'}`
- For adjectives, adverbs, prepositions, conjunctions, numerals, pronouns: just `word.part_of_speech`
- Use actual field values from JSON — check what part_of_speech values look like ("noun", "verb", "adjective", etc.)

RULES:
- Use textContent for Latin content — NOT innerHTML — so macrons render correctly
- Fetch path: `'data/vocabulary/all.json'` — no leading slash
- URLSearchParams decodes %2F automatically — do NOT manually decode
- Fully self-contained — no globals shared with app.js
- war/army links must use `vocabulary.html?topic=war%2Farmy` (encoded slash in href)

THEN: Read vocabulary.html. Add `<script src="js/vocabulary.js"></script>` after `<script src="js/app.js"></script>` and before `</body>`. Do NOT add to any other HTML page.
