<task type="auto">
  <name>Task 2: Create js/grammar.js — fetch and render all grammar tables</name>
  <files>js/grammar.js</files>
  <action>
FIRST: Read data/grammar/nouns.json and data/grammar/verbs.json to confirm the exact field names and structure before writing any rendering code. Do not assume field names.

Create js/grammar.js as a new, self-contained vanilla JS file. Do NOT modify js/app.js.

Required structure:

```javascript
'use strict';

document.addEventListener('DOMContentLoaded', () => {
  const app = document.getElementById('app');

  Promise.all([
    fetch('data/grammar/nouns.json').then(r => { if (!r.ok) throw new Error('nouns'); return r.json(); }),
    fetch('data/grammar/verbs.json').then(r => { if (!r.ok) throw new Error('verbs'); return r.json(); })
  ])
  .then(([nounsData, verbsData]) => {
    app.innerHTML = '';
    renderAllGrammar(app, nounsData, verbsData);
  })
  .catch(err => {
    app.innerHTML = '<h1>Grammar</h1><p class="error">Could not load grammar tables. Please reload.</p>';
    console.error('[CE Latin] Grammar load error:', err);
  });
});
```

Implement these functions (adapt field names to match the actual JSON structure you read):

**renderAllGrammar(app, nounsData, verbsData):**
- Append h1 "Grammar Reference" to app
- Append h2 "Noun Declensions" to app
- Iterate nounsData.declensions, call renderDeclensionSection(decl) and append each to app
- Append h2 "Verb Conjugations" to app
- Iterate verbsData.conjugations, call renderConjugationSection(conj) and append each to app
- Append h2 "Irregular Verbs" to app
- Iterate verbsData.irregular, call renderIrregularSection(verb) and append each to app

**renderDeclensionSection(decl):**
- Create div.grammar-section
- h3 with decl.name (e.g. "First Declension")
- p.grammar-example with example noun and meaning (read JSON for exact field names)
- Call renderNounTable(decl) and append result
- Return the div

**renderNounTable(decl):**
- Create div.table-scroll wrapping a table.grammar-table
- thead row: "Case" | "Singular" | "Plural"
- tbody: iterate decl.cases array — one row per case
  - Cell 1: case name with first letter capitalised, class="case-label"
  - Cell 2: decl.singular[caseName]
  - Cell 3: decl.plural[caseName]
- Return the wrapper div

**renderConjugationSection(conj):**
- Create div.grammar-section
- h3 with conj.name
- Iterate tenses (read JSON to confirm how tenses are structured — likely conj.tenses as an object with keys like "present", "imperfect", "perfect")
- For each tense: append p.tense-label with tense name, then append renderVerbTable(tenseData)
- Return the div

**renderVerbTable(tenseData):**
- Create div.table-scroll wrapping a table.grammar-table
- thead row: "Person" | "Singular" | "Plural"
- tbody: rows for 1st, 2nd, 3rd person
  - Use tenseData.singular["1st"], tenseData.plural["1st"] etc. (confirm exact keys from JSON)
- Return the wrapper div

**renderIrregularSection(verb):**
- Same structure as renderConjugationSection
- h3 with verb.name (e.g. "sum (to be)")
- Render each tense table

RULES:
- Use createElement/createTHead/createTBody/insertRow/insertCell for tables — NOT innerHTML for Latin forms (macrons must not be mangled by HTML entity encoding)
- Fetch paths: 'data/grammar/nouns.json' — no leading slash (GitHub Pages requires root-relative paths without leading /)
- No globals shared with app.js — fully self-contained
- 'use strict' at top
