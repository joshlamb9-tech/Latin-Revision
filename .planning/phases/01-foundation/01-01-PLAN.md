    Create the directory data/vocabulary/ and write data/vocabulary/all.json.

    Every entry MUST follow this exact schema:
    ```json
    {
      "id": "amicus-01",
      "latin": "amīcus",
      "genitive": "amīcī",
      "gender": "m",
      "english": "friend",
      "part_of_speech": "noun",
      "declension": 2,
      "conjugation": null,
      "topics": ["daily-life", "family"],
      "frequency_rank": 12
    }
    ```

    Field rules:
    - `id`: kebab-case latin lemma + zero-padded suffix (e.g., "porto-01", "via-01"). Suffix only needed if two entries share a root.
    - `latin`: nominative singular (nouns) or first person singular present (verbs). Include macrons as Unicode characters (ā ē ī ō ū Ā Ē Ī Ō Ū). Store as actual Unicode, not HTML entities.
    - `genitive`: genitive singular for nouns (null for verbs). Include macrons.
    - `gender`: "m", "f", "n", or null for non-nouns.
    - `english`: primary English meaning — one concise phrase.
    - `part_of_speech`: one of "noun", "verb", "adjective", "adverb", "preposition", "conjunction", "pronoun", "numeral".
    - `declension`: 1, 2, or 3 for nouns; null for others.
    - `conjugation`: 1 or 2 for verbs (CE syllabus covers 1st and 2nd conjugation only); null for others.
    - `topics`: array of strings from: ["family", "war/army", "gods", "travel", "daily-life", "nature"]. Assign at least one per entry. A word may appear in multiple topics.
    - `frequency_rank`: integer, 1 = most frequently examined. Assign based on prominence in specimen papers — words appearing in both Level 1 and Level 2 papers rank highest.

    For verbs, record:
    - `latin`: first person singular present (e.g., "portō" not "portare")
    - `genitive`: null
    - `gender`: null

    For adjectives that decline, record:
    - `latin`: masculine nominative singular
    - `genitive`: null (adjective paradigms are in grammar JSON, not vocab)

    The file must be valid JSON. Run a JSON parse check before completing. Aim for comprehensive coverage — every word that appears in either specimen paper's vocabulary list, plus common CE vocabulary Josh teaches.

    IMPORTANT: The file must be saved with UTF-8 encoding. Do not escape macron characters as \uXXXX — store them as literal Unicode.

    Encode 1st, 2nd, and 3rd declension noun paradigms. Schema:

    ```json

            "genitive": "puellae",

            "ablative": "puellā"
          },
          "plural": {
            "nominative": "puellae",
            "vocative": "puellae",
            "accusative": "puellās",
            "genitive": "puellārum",
            "dative": "puellīs",
            "ablative": "puellīs"

        }
      ]
    }
    ```

    Paradigms to include:
    - 1st declension: puella, puellae f (girl) — nominative/vocative/accusative/genitive/dative/ablative, singular and plural
    - 2nd declension (masculine -us): servus, servī m (slave) — all 6 cases, sg/pl
    - 2nd declension (neuter -um): bellum, bellī n (war) — all 6 cases, sg/pl
    - 3rd declension (masculine/feminine): rex, regis m (king) — all 6 cases, sg/pl
    - 3rd declension (neuter): nomen, nominis n (name) — all 6 cases, sg/pl

    All forms must include macrons where standard. Use authoritative forms from standard Latin grammars (Kennedy's Latin Primer).
